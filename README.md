# Study Command Center v.2

Study Command Center v.2 is a personal coding progress dashboard built with HTML and CSS. The project tracks current learning focus, completed work, review topics, project progress, resources, and next steps in a styled command-center layout.

This version redesigns the original Learning Dashboard into a more polished mini control panel with a darker interface, dashboard-style cards, status indicators, glowing accents, and a more intentional visual structure.

## Overview

The goal of this project was to create a visual dashboard that organizes my coding progress in one place. Instead of using a plain notes-style layout, Study Command Center v.2 presents the information like a personal developer control panel.

The dashboard includes sections for:

- System status
- Weekly briefing
- Completed tasks
- Debug queue
- Build timeline
- Reference library
- Active modules
- Next focus

## Features

- Dark command-center inspired layout
- Responsive dashboard grid
- Featured top-row status panels
- Smaller supporting cards arranged in a centered card layout
- Glowing card borders and hover effects
- Animated status indicator dots on card hover
- Badge-style labels for current learning information
- Custom typography using Google Fonts
- Decorative star images positioned in the header panel
- Mobile-friendly layout with responsive media queries

## Built With

- HTML
- CSS
- CSS Grid
- Responsive design
- Google Fonts

## What I Built

I built a personal study dashboard that tracks my current coding progress and learning priorities. The project uses a structured card layout to organize information into clear sections, making it easy to see what I am learning, what I have completed, what I need to review, and what I plan to work on next.

The second version focuses more heavily on visual polish. I redesigned the project from a soft learning dashboard into a darker control-panel style interface with stronger spacing, glowing borders, system-style labels, and a more intentional layout.

## Major Sections

### System Status

Shows the current learning phase, current focus, active project, and progress status.

### Weekly Briefing

Summarizes the main learning priorities and concepts being practiced during the week.

### Completed Tasks

Highlights recent coding milestones and finished work.

### Debug Queue

Lists concepts that need more review or practice.

### Build Timeline

Tracks completed and active projects.

### Reference Library

Lists learning resources and documentation used while studying.

### Active Modules

Shows the current JavaScript topics being studied.

### Next Focus

Lists upcoming learning goals and project planning steps.

## Design Direction

The design direction for Study Command Center v.2 is a dark, futuristic dashboard. The layout is meant to feel like a small developer control panel rather than a basic project page.

The design uses:

- A dark gradient background
- Neon-inspired accent colors
- Glowing card borders
- Rounded dashboard panels
- Colored status lights
- Badge elements
- A structured grid layout
- Decorative corner stars in the header

The overall goal was to make the project feel more polished, more intentional, and more portfolio-ready.
