# Learning Dashboard

A soft, colorful learning dashboard built with HTML and CSS. This project tracks current study progress, weekly goals, recent wins, review topics, resources, and upcoming project milestones.

## Overview

The Learning Dashboard is a personal front-end project designed to organize and display coding progress in a visually engaging way. It uses a playful purple and blue color palette, custom fonts, badges, hover effects, decorative stars, and responsive card layouts.

This project was created as part of my ongoing web development practice while studying HTML, CSS, Flexbox, responsive design, and visual styling.

## Features

- Responsive card-based dashboard layout
- Decorative header with star images
- Current learning status section
- Weekly focus and completed lessons
- Recent wins tracker
- Review topics section
- Project timeline
- Resource shelf
- Study topics list
- Custom badges
- Custom list bullets
- Link hover, active, visited, and focus states
- Mobile-friendly media query

## Built With

- HTML
- CSS
- Google Fonts

## What I Practiced

Through this project, I practiced:

- Structuring a webpage with semantic HTML
- Styling reusable card sections
- Using Flexbox for layout
- Creating hover effects with `transform`
- Adding responsive behavior with media queries
- Styling links for multiple states
- Creating custom badges with `span` elements
- Using custom list bullets with `::before`
- Working with background images and layered backgrounds
- Improving visual hierarchy with typography, spacing, borders, and shadows
